Setting Up the Development Environment
--------------------------------------
- Install Node
    - Download and install the latest LTS (Long Term Support) version

    - Use the Node Package Manager (npm) 
        - To install the TypeScript compiler
        - As well as other 3rd party packages
    
    - Website (https://nodejs.org/en)


- To install the TypeScript compiler (globally) via npm on terminal

    sudo npm i -g TypeScript


- Post installation check on terminal

    tsc -v


- Install VS Code
    - Preferred code editor
    - Website (https://code.visualstudio.com/)
