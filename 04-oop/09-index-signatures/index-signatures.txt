Index Signatures
----------------
- Index signatures allows TypeScript
    - To define a property in a class
        - That will allow to dynamically append new properties

    - Similar to that of JavaScript
        - Except with the benefit of "Type Checking" c/o TypeScript


- Dynamically append new properties to an object using
    - The "." notation
    - The "[]" notation


Syntax:

    class <class_name> {
        [<property_name>: <data_type>]: <data_type>;
        ...
    }

    let <object_name> = new <class_name>();
    
    <object_name>.<new_property_name> = <assigned_value>;
    <object_name>[<new_property_name>] = <assigned_value>;
